# Binding of Mech

一款基于 HTML5 Canvas 的俯视角二足机甲动作射击游戏。玩家驾驶可高度自定义的机甲在随机生成的战场中执行任务，击败敌人、获取资源、升级机体与武器，并体验“机甲被毁后驾驶员出舱继续作战”的独特机制。

## 游戏概述

### 核心体验

- **双形态作战**：玩家既可以在重型机甲中横扫战场，也可以在机甲被毁后弹出舱，以单兵形态继续战斗。
- **Roguelike 成长**：通过任务获取金币与经验，在格纳库中升级装甲、引擎、结构和冲刺系统，并解锁/强化多种武器。
- **高度可编辑**：内置武器工坊、杂兵编辑器和关卡编辑器，玩家可以自由调整武器参数、设计敌人类型、创建关卡。

### 世界观（简要）

在资源枯竭的未来世界，各方势力驾驶着被称为“二足机甲”（Bipedal Mech）的步行兵器争夺最后的生存地带。玩家作为一名雇佣机甲驾驶员，接受各种高危险任务，在废墟、森林与人工障碍之间与敌方单位交战。

## 核心玩法

### 机甲操作

| 按键 | 功能 |
|------|------|
| `W` / `S` | 前进 / 后退 |
| `A` / `D` | 左右侧移 |
| `Q` / `E` | 机甲转向 |
| `←` / `→` | 武器/炮台转向 |
| `空格` | 射击 / 近战挥舞 |
| `1-5` | 切换武器 |
| `F` | 发射钩爪 |
| `Shift` + 方向 | 冲刺 |
| `X` | 驾驶员弹出舱 |
| `G` | 靠近机甲时重新驾驶 |
| `R` | 使用修理包 |
| `Esc` | 退出任务 |

### 驾驶员单兵操作

| 按键 | 功能 |
|------|------|
| `W` / `A` / `S` / `D` | 移动 |
| 鼠标 | 瞄准 |
| `空格` | 小手枪射击 |
| `G` | 靠近机甲时重新驾驶 |

### 武器系统

| 武器 | 特点 |
|------|------|
| 火神炮 | 高射速连发，射击会积累热量，过热后需冷却 |
| 霰弹枪 | 近距离散射，多发弹丸 |
| 加农炮 | 高伤害单发，弹道爆炸效果 |
| 激光炮 | 即时命中光束，穿透敌人 |
| 光束刀 | 近战武器，挥舞时产生光刃斩击 |

### 杂兵种类

| 类型 | 特点 |
|------|------|
| 侦察兵 | 速度快，血量低 |
| 步兵 | 基础单位，均衡属性 |
| 重装兵 | 高血量，使用加农炮 |
| 狙击手 | 远程激光，高威胁 |
| 霰弹手 | 近距离高爆发 |
| 医疗兵 | 绿色标识，支援型 |

### 状态效果

- **着火**：受击有概率进入着火状态，持续掉血并产生火焰粒子。
- **冒烟**：生命值低于 60% 开始冒烟，低于 30% 冒烟更浓密。
- **碰撞伤害**：机甲之间发生碰撞时，根据相对速度产生伤害。

### 关卡与奖励

- 每个关卡在 2400×2400 的地图上随机生成障碍物、可破坏树木和敌人。
- 击败敌人掉落金币袋（30% 概率掉落修理包）。
- 完成任务后获得金币和经验奖励，用于格纳库升级。

## 编辑系统

### 武器工坊

- 调整伤害、射速、散布、弹速、弹径等参数。
- 支持导入/导出 JSON 配置。

### 杂兵编辑器

- 在主菜单进入，可修改现有杂兵属性或新增自定义类型。
- 支持导入/导出 JSON 配置。

### 关卡编辑器

- 设计关卡编号、名称、敌人数量、敌人类型、奖励等。
- 支持导入/导出 JSON 配置。

## 未来改进方案

### 短期优化

1. **音效系统**
   - 为射击、爆炸、命中、冲刺、机甲移动等添加音效。
   - 背景音乐随战斗强度变化。

2. **粒子与视觉反馈增强**
   - 更丰富的爆炸、命中、燃烧粒子效果。
   - 屏幕震动、受伤红屏等反馈。
   - 机甲不同血量阶段的外观变化（破损、冒烟、起火）。

3. **操作手感优化**
   - 降低机甲转向惯性，提供更灵敏的响应。
   - 优化鼠标瞄准与炮台转向的阻尼感。
   - 改进冲刺后的硬直与按键冲突问题。

4. **Bug 修复与稳定性**
   - 修复钩爪偶尔出现的负半径渲染错误。
   - 优化大量敌人时的性能。
   - 解决模块循环引用问题。

### 中期内容扩展

1. **更多武器类型**
   - 导弹发射器：追踪导弹，对重甲有效。
   - 电磁脉冲：范围眩晕机械单位。
   - 狙击步枪：超远距高精度单发。
   - 火焰喷射器：持续火焰喷射，造成燃烧效果。

2. **更多敌人类型**
   - 自爆单位：靠近玩家后引爆。
   - 飞行单位：可跨越障碍，需要特殊武器应对。
   - 精英/BOSS 单位：拥有多阶段战斗机制。
   - 医疗兵 AI：实际为友方或敌方单位提供治疗。

3. **环境互动**
   - 可破坏的油桶、箱子，爆炸后对周围造成伤害。
   - 草丛、烟雾弹提供隐蔽。
   - 陷阱与地雷。

4. **任务类型多样化**
   - 护送任务：保护友方单位到达目标点。
   - 生存任务：抵御多波敌人进攻。
   - 据点防守：占领并防守特定区域。
   - BOSS 战：特殊的大型机甲或要塞。

5. **格纳库扩展**
   - 机甲涂装与外观自定义。
   - 装备模块系统：不同模块提供特殊能力（雷达、自动修复、能量盾）。
   - 驾驶员技能树：提升单兵作战能力。

### 长期发展方向

1. **多人联机**
   - 本地合作模式（2P 同屏）。
   - 在线 PvE 合作任务。
   - PvP 竞技场模式。

2. **程序生成关卡**
   - 基于种子生成无限关卡。
   - 不同类型的生态区域（废墟、沙漠、雪地、森林）。

3. **剧情与 Roguelike 模式**
   - 加入剧情战役，解锁新的机甲、武器、敌人。
   - 类 Roguelike 的“一条命挑战”模式。

4. **Mod 支持**
   - 更完善的 JSON Mod 导入导出。
   - 允许玩家通过脚本扩展武器、敌人、关卡。

5. **移植与发布**
   - 打包为桌面应用（Electron / Tauri）。
   - 移动端触屏适配。
   - 上架 itch.io 等独立游戏平台。
